## Project Overview
This project implements a scalable data pipeline using Snowflake.  
Data is ingested from CSV and JSON files into a staged architecture consisting of RAW, VALIDATED, and CURATED layers.  

The pipeline performs data ingestion, validation, transformation, and incremental processing using MERGE logic. Security and governance are implemented using RBAC, Dynamic Data Masking, and Row Access Policies. Automation is achieved through Streams and Tasks for efficient data processing and KPI-ready analytics tables.
